Description du Projet

Développement d'un site web statique moderne et responsive pour l'agence VITRINE CONSULTING. Le site comprend une vitrine professionnelle avec contenu dynamique via JavaScript, galerie média interactive, présentation des services et projets, formulaire de contact, système multilingue (FR/EN/DE), et une interface utilisateur optimisée pour tous les appareils. Technologies utilisées : HTML5, CSS3 avec variables CSS, JavaScript vanilla, Google Fonts.
